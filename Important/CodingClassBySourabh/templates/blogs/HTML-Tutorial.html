A tags is like a container for either content or other HTML tags.

HTML Language is a Case Sensitivity language. For Example <H1> and <h1> are Same Tag.

Tags

Tags are two types:

1) Opening Tag

1) Closing Tag

Important Notes: Some are self closing tag. Like → <img />, <br />, <hr /> etc.

Opening Tag

Opening tag are display in <TagName>

Closing Tag

Closing tag are display in </TagName>

Comments In HTML

Comment in HTML are used to mark text which should not be parsed. They can help Document the Source Code

Comment are Display in <!-- Comments In HTML -->

HTML used attributes to add more Information in Corresponding to a HTML tags.

For Example:

Container Tags

Div Tag

Div tag is often used as a container for other elements. div is a block container tag. It takes full width 

<div>Content and Other tags</div>

Span Tag

Span tag is a container tag for other elements. Span is a inline container tag. It takes as much width as necessary. 

<span>Content and Other tags</span>

Table

A table is a collection of rows and columns. It is used to represent data in tabular form.

Table of Structure

<table>
    <caption>Table</caption>
    <thead>
        <tr>
            <th>First Column>/th>
            <th>Second Column>/th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
            <td>Data 3</td>
        </tr>
        <tr>
            <td>Data 1st</td>
            <td>Data 2nd</td>
            <td>Data 3rd</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>&nbsp;</td>
            <td>Data</td>
            <td>Data</td>
        </tr>
    </tfoot>
</table>
